Summary: Latitude vs. 'X'

UCSD Bootcamp

The purpose of this project is to analyze the impact on weather based on the location relative to the equator. To understand the possible impact, we pulled data from 500+ sites from OpenWeatherMap API as a starting dataset. After assembling the data, we used Python, Pandas and Matplotlib to plot 4 different weather attributes realtive to the latitude

This site provides the source data and the vizualizations created as part of the analysis, as we ll as explanations and descriptions of any trends and correlations witnessed
